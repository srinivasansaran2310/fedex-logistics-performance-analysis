# fedex-logistics-performance-analysis
Exploratory Data Analysis of FedEx shipment and logistics data to identify delivery delays, freight cost patterns, vendor performance, and supply chain optimization opportunities.

## Project Overview

This project analyzes FedEx shipment and logistics data to identify delivery delays, freight cost patterns, shipment trends, and vendor performance. The objective is to improve logistics efficiency and support data-driven decision-making.

## Business Objective

- Improve delivery efficiency
- Reduce shipment delays
- Optimize transportation costs
- Enhance vendor performance
- Improve customer satisfaction

## Dataset Information

- Total Rows: 10,324
- Total Columns: 33
- Duplicate Records: 0
- Missing values handled through data preprocessing

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning & Preparation

- Checked duplicate records
- Handled missing values
- Converted date columns into datetime format
- Converted numerical columns into numeric format
- Created Delivery Lead Time KPI
- Created Delayed Shipment Status KPI

## Exploratory Data Analysis

The following analyses were performed:

1. Shipment Mode Distribution
2. Delayed vs On-Time Shipments
3. Delivery Lead Time Distribution
4. Monthly Shipment Trends
5. Freight Cost Analysis
6. Correlation Heatmap
7. Pair Plot Analysis

## Key Insights

- Air shipment mode was the most frequently used transportation method.
- Most shipments were delivered on time.
- Freight costs increased with shipment weight.
- Delivery lead times varied across shipment modes.
- Some shipment routes and vendors showed higher delays.

## Business Recommendations

- Optimize shipment mode selection based on cost and urgency.
- Improve vendor performance monitoring.
- Reduce delivery delays through better route planning.
- Use KPI tracking for continuous logistics improvement.

## Conclusion

The analysis identified opportunities to improve delivery performance, reduce transportation costs, and enhance supply chain efficiency. The findings support better operational planning and data-driven decision-making for FedEx logistics operations.
